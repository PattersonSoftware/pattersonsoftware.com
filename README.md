# PattersonSoftware.com

This is the source code to the site located at pattersonsoftware.com.

## Tech Stack / System Requirements

* Node 25
* React 19.2
* Vite 7.2
* TypeScript 5.9
* Tailwind CSS 4.2

Please see package.json for any specifics about additional dependencies or command definitions.

## Running Locally

To run this code locally, you must have Node 24 with NPM already installed. Then, run `npm install` in the root
directory, which should install all dependencies. To actually run the site, simply run `npm run dev` which will start
Vite with HMR for any changes that are made on the site.
